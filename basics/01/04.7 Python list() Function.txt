The list() function creates a list object.
A list object is a collection which is ordered and changeable.


The list() Constructor
It is also possible to use the list() constructor when creating a new list.>>> Room1 =list(('Vishnu','Thejus','AlanBenny','Abhilash'))


>>> print(Room1)
['Vishnu', 'Thejus', 'AlanBenny', 'Abhilash']
